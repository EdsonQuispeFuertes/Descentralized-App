This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`]

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Use the DApp

To be able to use the DApp you can install Ganache here (https://trufflesuite.com/ganache/)

And download the extension Metamask un your browser, with the extension and the accounts of Ganache you can use this DApp.
